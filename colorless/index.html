Colorless+JSON


The mapping of Colorless+JSON into transported JSON is predictable.

Wrapper

{ "n": "Id", "w": "String" }
"SomeId"

Struct

{ "n": "Coordinate", "m": [ {"x":"F32"}, {"y":"F32"} ] }
{ "x": 1.0, "y": 2.0 }

Enum

{ "n": "Color", "e": [
        "Red",
        "Blue",
        "Green",
        { "n": "Custom", "m": [ {"red":"U8"}, {"green":"U8"}, {"blue":"U8"} ] }
    ]
}
{ "n": "Red" }

{ "n": "Blue" }

{ "n": "Green" }

{ "n": "Custom", "m": { "r": 234, "g": 240, "b": 10 } }

Union

{ "n": "Stuff", "u": [
        "Id",
        "Coordinate",
        "Color"
    ]
}
{ "n": "Id", "w": "SomeId" }

{ "n": "Coordinate", "m": { "x": 1.0, "y": 2.0 } }

{ "n": "Color", "e": { "n": "Blue" } }

All types double as functions.

{ "n": "Hello", "m": [ {"target":"String"} ], "o": "String" }
{ "target": "World" }
"Hello, World!"
